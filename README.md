# 👨‍💼 Employee Manager System

A full-stack web application for managing employee records, built using **Java Spring Boot** and **MySQL**, with a simple and responsive frontend.

---

## 🚀 Features
- ➕ Add new employee details  
- ✏️ Update employee information  
- ❌ Delete employee records  
- 📋 View all employees in a structured list  
- 🔍 Simple and clean UI for easy management  

---

## 🛠️ Tech Stack

### Backend
- Java (OOPs, Core Java)
- Spring Boot
- Spring MVC
- Spring Data JPA (Hibernate)
- REST APIs

#Database
- MySQL

# Frontend
- HTML5
- CSS3
- JavaScript
